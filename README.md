# 百科全书

> 当前只有经济学概念笔记——以曼昆《经济学原理》为骨架，范里安《微观经济学：现代观点》、Mishkin《货币金融学》、果冻AP经济学为延伸。

---

## 宏观经济学

`经济学/宏观经济学/` — 17 篇 + MOC

| 主题 | 笔记 |
|------|------|
| 基础度量 | GDP与经济福利、物价与生活成本 |
| 长期真实经济 | 生产率与增长、储蓄投资与金融体系、基本金融工具、失业 |
| 货币与物价 | 货币与中央银行、货币增长与通货膨胀、货币政策传导机制 |
| 开放经济 | 国际贸易与汇率、资本流动与金融危机 |
| 经济史比较 | **大萧条与现代金融危机比较**（新增） |
| 短期波动 | 总需求与总供给、财政与货币政策、通胀与失业的权衡 |
| 金融体系深度 | **金融结构的经济学分析**（信息不对称/柠檬问题，新增） |

## 微观经济学

`经济学/微观经济学/` — 26 篇 + MOC

| 模块 | 笔记 |
|------|------|
| 地基概念 | 稀缺性、机会成本、边际思维、激励 |
| 交换与市场 | 比较优势、供给与需求、弹性、价格管制与税收 |
| 企业行为 | 生产成本、规模经济 |
| 市场结构 | 完全竞争、垄断、垄断竞争、寡头、博弈论、价格歧视 |
| 市场失灵 | 外部性、公共品与搭便车 |
| 消费者与要素 | 消费者选择理论、生产要素市场 |
| **延伸主题** | **跨期选择不确定性与风险**、**行为经济学**、**信息技术与网络外部性** |

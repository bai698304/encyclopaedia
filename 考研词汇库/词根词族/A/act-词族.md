# act-词族

> 核心词: **[[act]]** — v.表演;举动;起作用 n.行为,法令;一幕
> 词根 act/ag = 做,驱动

共 **10** 个词

## 核心派生链

> 后缀逐步派生

**[[act]]** v./n. 行动;表演
  -> *+ive a....的* -> **[[active]]** a. 活跃的,积极的
  -> *+ate v./a.使成为* -> **[[activate]]** v. 启动,激活
  -> *+ity n.性质,状态* -> **[[activity]]** n. 活动;活力
  -> *+or n.人/物* -> **[[actor]]** n. 演员,行动者
  -> *+ual a....的* -> **[[actual]]** a. 实际的,真实的

## 前缀扩展

> 不同前缀 + 词根 → 不同含义

### ex- 出,外,前 (1词)

- **[[exact]]** a. 确切的,精确的 *ex(出)+act(做) → 做出来 → 精确的*

### inter- 之间,相互 (1词)

- **[[interact]]** v. 互相作用,互相影响 *inter(相互)+act(作用)*

### re- 再,又,回 (1词)

- **[[react]]** v. 反应,起作用 *re(回)+act(做) → 做回去 → 反应*

# port-词族

> 核心词: **[[port]]** — n.港口
> 词根 port = 携带,运送,门

共 **11** 个词

## 核心派生链

> 后缀逐步派生

**[[port]]** n.港口;携带
  -> *+able a.可...的* -> **[[portable]]** a. 轻便的,手提式的
  -> *+er n.人/物* -> **[[porter]]** n. 行李搬运工,门房

## 前缀扩展

> 不同前缀 + port → 不同含义；缩进项为后缀再派生

### ex- 出,外,前 (1词)

- **[[export]]** v./n. 输出,出口 *ex(出)+port(运) → 运出去*

### im- 向内 (3词)

- **[[import]]** v./n. 进口,输入 *im(入)+port(运) → 运进来*
  - **[[importance]]** n. 重要,重要性 *(+ance n.性质,状态)*
  - **[[important]]** a. 重要的,有地位的 *(+ant a./n.人)*

### re- 再,又,回 (2词)

- **[[report]]** n./v. 报告,汇报 *re(回)+port(带) → 带回来 → 报告*
  - **[[reporter]]** n. 记者,通讯员 *(+er n.人/物)*

### trans- 跨越,转移 (1词)

- **[[transport]]** v./n. 运输,运送 *trans(跨越)+port(运)*

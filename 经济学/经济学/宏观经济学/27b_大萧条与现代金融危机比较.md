# 大萧条与现代金融危机比较

> **大萧条是宏观经济学的"圣杯"——理解了大萧条，就理解了危机为什么发生、为什么不同危机的结果截然不同。** —— Bernanke (1995)

## 内涵与外延

### 一句话定义

**大萧条(1929-1933)是历史上最严重的全球经济危机，核心特征是世界货币供给崩溃+金本位传导+债务通缩。比较大萧条与现代金融危机（2008、亚洲、欧债），可以揭示决定危机深度的关键变量。**

### 大萧条的核心机制——三大因素(Bernanke 论文)(V)

Bernanke 的比较研究发现，大萧条不是单一的美国故事，而是通过金本位传导的全球性货币崩溃：

**1. 金本位——"好"制度的致命缺陷**

1920s 各国回归金本位，但金本位将各国货币供给锁定在黄金储备上→一国危机导致黄金外流→另一国货币被迫紧缩→紧缩通过金本位传导到全球。Bernanke 发现**1931年奥地利信贷银行(Kreditanstalt)倒闭后**，货币危机从一个小国开始，通过金本位+银行恐慌+汇率危机三股力量席卷全球。**货币乘数崩溃**——银行恐慌导致存款-现金比率上升，同一单位基础货币能支撑的货币供给大幅下降。

**2. 通缩——债务通缩螺旋(伯南克1983论文)**

物价持续下跌→企业名义债务不变但实际债务加重（卖产品换回来的钱变少了）→企业违约→银行坏账→信贷冻结→进一步压物价。这是三阶段模型第三阶段（债务通缩）的唯一完整案例。

Bernanke 强调：**金融合约的非指数化（nominal non-indexation）是关键**——债务合约是以名义金额签订的，物价下跌不会自动减少还款额。如果合约被指数化了（随物价变动自动调整），债务通缩就不会如此严重。

**3. 名义工资粘性**

物价下跌后名义工资调整缓慢→实际工资上升→企业雇不起人→失业飙升（美国失业率最高达25%）。Bernanke 认为名义工资粘性是货币冲击影响实体经济的第二个渠道。

### 大萧条 vs 2008 全球金融危机——决定性差异

Bernanke 论文和 Mishkin Ch12 分别提供了两场危机各自的分析框架，对比如下：

| 维度 | 大萧条(1929-1933) | 2008 全球金融危机 |
|------|------------------|------------------|
| **触发** | 股市崩盘+银行恐慌→货币供给崩溃 | 次贷违约→MBS/CDO 暴跌→雷曼破产 |
| **货币框架** | 金本位——各国央行被动紧缩 | 法定货币——央行可主动扩张 |
| **银行危机** | 大量银行倒闭（美国超9000家） | 大银行被救助（除雷曼外） |
| **三阶段进展** | 进入第三阶段：债务通缩 | 停在第二阶段：及时干预 |
| **美联储行动** | 被动（紧缩货币保护金本位） | 主动（降息至零+QE+救助） |
| **财政政策** | 胡佛增税+减少支出（紧缩） | 大规模财政刺激+TARP 救助 |
| **失业率峰值** | ~25% | ~10%（美国） |
| **持续时长** | 4年衰退+多年缓慢复苏 | 18个月衰退（2007.12-2009.6） |

**核心结论(Bernanke + Mishkin)：** 2008 年之所以没有重演大萧条，不是因为 2008 的冲击更小——而是因为美联储学会了"在危机中不要重复 1930s 的错误"。具体来说：

1. **央行必须果断做最后贷款人**——大萧条中美联储没有履行这个职责，2008 中伯南克做到了
2. **不要固守错误的货币制度**——大萧条中各国死守金本位，被迫紧缩；2008 中法定货币给了央行无限扩张空间
3. **财政政策不能紧缩**——胡佛增税加深了萧条，2008 各国同步财政刺激
4. **必须防止债务通缩螺旋**——一旦物价和资产价格开始螺旋下跌，必须零利率+大规模注入流动性打断循环

### 亚洲金融危机(1997)——新兴市场的版本

亚洲金融危机与美欧危机的差异同样说明问题：

| 维度 | 亚洲(1997) | 2008(美国) |
|------|-----------|-----------|
| 触发 | 固定汇率+大量短期外债→货币攻击 | 国内资产泡沫+金融创新失控 |
| 核心矛盾 | 货币错配（借美元/还本币） | 信息不对称（MBS 真实风险不可知） |
| 国际货币基金组织介入 | 条件苛刻（紧缩+加息）→加深衰退 | 无类似 IMF 角色 |
| 恢复速度 | 2-3年（韩国较快，印尼缓慢） | ~4年（但未出现大萧条式持续萧条） |

小Lin说"资本是如何接连引爆"(2026)(V)完整复盘了泰国→韩国→印尼→马来西亚的传染链。**亚洲危机的核心教训是：开放资本账户+固定汇率+短期外债 = 危机定时炸弹。**

### 欧债危机(2010-2012)——货币联盟内的不对称危机

希腊、西班牙、爱尔兰的债务危机与前述类型都不同：它们没有自己的货币→不能贬值刺激出口→不能独立执行货币政策→唯一的调整途径是内部贬值的紧缩（降工资+减福利+增税）→政治崩溃风险。小Lin说欧债危机(2022)(V)完整覆盖了这条逻辑链。

### 常见误解

- ❌「大萧条是1929股市崩盘导致的」→ ✓ 股市崩盘只是起点。真正的原因是世界货币供给崩溃（金本位）加上政策错误（央行紧缩、胡佛增税）。
- ❌「2008 没有重演大萧条是因为我们学了教训」→ ✓ 确实如此。但 Bernanke 本人就写过大萧条论文，他2006年接任美联储主席时将学术研究直接转化为政策行动——这是"学术影响政策"最直接的实证。
- ❌「金融危机都一样，只是程度不同」→ ✓ 金本位危机、银行危机、货币联盟危机、汇率危机的机制完全不同。适合希腊的政策（紧缩）可能彻底搞垮阿根廷。

## 实战中的反复出现模式

### 模式 1：同一个央行行长，同一个学术信念——Bernanke 与大萧条

Bernanke 作为普林斯顿教授时大量研究大萧条。1995 年他写道"理解大萧条是宏观经济学的圣杯"(V)。2006 年他成为美联储主席→2008 年直面危机→他的政策决策直接基于其学术研究：他不让银行系统崩溃（最后贷款人）、不让货币供给收缩（QE）、不让债务通缩发生（零利率+通胀目标）。**这是宏观经济理论影响真实命运的最完整案例——如果 2008 年美联储主席是一个不了解大萧条的人，结果可能完全不同。**

### 模式 2：金本位的约束 vs 法定货币的自由——不同制度决定不同命运

1931 年奥地利 Kreditanstalt 银行倒闭→投资者恐慌→多国银行挤兑→公众提取黄金→各国央行提高利率保护黄金储备→货币紧缩→萧条加深。没有人想让萧条更严重——但金本位下的所有理性行动都指向了同一个灾难。相比之下，2008 年美联储降息至零+购买 MBS+提供紧急贷款——法定货币制度给央行提供了大萧条中完全没有的政策空间。**制度约束比意图更重要。**

### 模式 3：小Lin说各危机视频——同一分析框架的不同应用

小Lin说的危机系列（希腊、亚洲、英国、2008）有一个共同的叙事结构：**识别失衡→不可持续→引爆点→恐慌传染→政策应对**。这个框架与 Mishkin 的三阶段危机模型高度吻合。区别在于每次危机的"引爆点"不同（希腊→债务到期、亚洲→汇率投机、英国→财政信誉崩塌），但核心动力（信心瓦解→资本逆转→恐慌传染）完全相同。(V)

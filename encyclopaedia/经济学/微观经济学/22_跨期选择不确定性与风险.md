# 跨期选择、不确定性与风险

> **现实中的大多数决策涉及两个维度：时间（今天花钱还是存着明天花？）和不确定性（投资这只股票可能赚钱也可能亏本）。标准消费者理论假设同时性和确定性——这里我们放松这两个假设。** —— 范里安《微观经济学：现代观点》第 10-13 章

## 内涵与外延

### 跨时期选择——今天花还是明天花(范里安 Ch10)(V)

人们面临的核心跨时期决策：**当前消费 vs 储蓄（未来消费）。** 预算约束：你今天花的钱 + 存下来明天花的钱（折现值）= 你的终生资源。

**关键概念：现值**
- 未来 1 美元的现值 = 1/(1+r)^t。利率 r 越高，未来的钱越不值钱
- 这个简单公式是一切投资决策的基础：企业投资项目、个人买房贷款、政府评估政策

**利率对消费决策的影响：**
- 利率上升 → 储蓄的回报更高 → 今天少花、明天多花（替代效应）
- 利率上升 → 利息收入增加 → 可能今天就多花（收入效应）
- 最终结果取决于哪种效应占主导

### 资产市场——套利与泡沫(范里安 Ch11)(V)

**套利(arbitrage)**：同一资产在不同市场上有不同价格 → 低价买入、高价卖出 → 赚取无风险利润。套利行为推动价格回归一致——这是金融市场最重要的均衡力量。

**市场泡沫**：当资产价格持续偏离其基本面价值（未来收益的现值），就形成了泡沫。范里安指出：**泡沫可以存在，前提是没有人能确定当前价格是错的——或者明知道是泡沫但相信"更大的傻瓜"会接手。** 荷兰郁金香、2000 年互联网泡沫、2008 年房地产泡沫都遵循这个逻辑。

> 有效市场假说认为泡沫不会长期存在（套利会消灭它们）。但行为经济学（#31）和实际市场历史都表明：泡沫是真实存在的——因为套利需要"卖空"被高估的资产，而卖空有成本和风险。

### 不确定性——期望效用理论(范里安 Ch12)(V)

当结果不确定时，人们如何做决策？标准模型：**期望效用最大化。**

**或有消费(contingent consumption)**：你的消费取决于某个不确定事件的结果。买了火灾保险 → 如果发生火灾，保险公司赔你钱，你的消费不会大幅下降。

**期望效用公式：** 如果事件以概率 p 产生结果 A，以概率 (1-p) 产生结果 B，那么决策者选择的依据是：

**EU = p × U(A) + (1-p) × U(B)**

不是最大化期望金额 p×A + (1-p)×B，而是最大化期望**效用**。

### 风险态度

| 类型 | 定义 | 例子 | 效用函数形状 |
|------|------|------|------------|
| **风险厌恶** | 偏好确定的结果而不是有相同期望值的赌博 | 买保险 | 凹函数（边际效用递减） |
| **风险偏好** | 偏好赌博而不是有相同期望值的确定结果 | 买彩票 | 凸函数（边际效用递增） |
| **风险中性** | 只在期望值之间做判断 | 统计决策 | 线性函数 |

> 大多数人在大多数决策中是风险厌恶的——这就是为什么保险行业能存在。但同样的一个人在买彩票时又表现出风险偏好——这说明风险态度不是单一维度的。

### 资产多样化（范里安的"不要把所有鸡蛋放在一个篮子里"）(范里安 Ch12-13)(V)

通过持有多种互不相关的资产，可以在不降低期望收益的情况下降低总体风险。这来自一个数学事实：**当两个随机变量不完全正相关时，组合的方差小于单个方差的加权和。**

| 概念 | 含义 |
|------|------|
| **资产多样化** | 分散投资于不同资产→降低非系统性风险 |
| **风险分散** | 把风险分摊到更多人身上（保险、股票市场） |
| **系统性风险** | 整个市场一起跌（2008）→多样化无法消除 |

### 常见误解

- ❌「买保险不划算（期望值上是亏的）」→ ✓ 保险买的不是期望值，是**确定性**。风险厌恶的人愿意支付溢价来消除不确定性。
- ❌「高收益=高风险」→ ✓ 理论上成立，但现实中存在被错误定价的资产——泡沫时期"低风险高收益"的错觉是最危险的。
- ❌「股票长期一定涨」→ ✓ 系统性风险无法通过长期持消除——日本股市 1990 年至今仍未回到高点。

## 实战中的反复出现模式

### 模式 1：保险——期望效用理论的最佳演示

你不买保险 → 你承担全部风险（房子烧了 → 损失 100 万）。你买保险 → 每年付保费 1000 元 → 无论是否发生火灾 → 你的消费稳定。付保费 = 用确定的小损失换取不确定的大损失的消除。期望效用理论精确预测了这种行为：**凹效用函数意味着失去 100 万的痛苦 > 获得 100 万的快乐**——因此规避风险是理性的。

### 模式 2：比特币——投机、泡沫与"更大的傻瓜"

比特币不产生现金流（没有股息、没有租金），因此无法用现值模型定价。它的价格完全取决于"下一个买家愿意出多少钱"——也就是**博傻理论(greater fool theory)**：只要能在崩盘前卖给"更大的傻瓜"，再高的价格也可以接受。范里安关于资产泡沫的分析框架在这里完美适用。

### 模式 3：美国 401(k) 退休储蓄——跨时期选择的实际应用

标准的代际消费模型预测：年轻人应该多储蓄（为退休），老年人应该花储蓄。但实际上大多数人储蓄不足——原因不是"理性不足"，而是即时满足的偏好（当前消费的效用 > 未来消费的贴现效用）。行为经济学的默认选项（#31）和跨时期选择理论结合起来解释了为什么自动加入 401(k) 计划如此有效。

# 信息技术与网络外部性

> **信息革命改变的不只是技术——它还改变了经济运行的逻辑。网络外部性、锁定效应、双边市场——这些概念解释了为什么一家公司可以免费提供产品却赚数十亿美元。** —— 范里安《微观经济学：现代观点》第 36 章

## 体系定位

- **前置**：[[10_规模经济|规模经济]]（需求侧规模经济的极端形式）；[[16_外部性|外部性]]（网络外部性是外部性的一种特殊形式）
- **后继**：无
- **所属分支**：微观 / 信息经济
- **来源原理**：当一种商品的价值取决于消费该商品的其他人的数量时，标准供需模型的分析需要修正——这不是生产端的规模经济，而是需求端的规模经济

## 内涵与外延

### 一句话定义

**网络外部性(network externalities) = 一个人从某种商品中获得的效用取决于消费该商品的其他消费者的数量。传真机、电话、社交软件都属于这种商品——你的传真机只有在别人也有传真机时才有用。**

### 系统竞争

信息技术产品通常是**系统**——由多个互补元件组成，不同厂商各自供应一部分。经典案例(范里安 Ch36)(V)：
- 硬件 + 软件：计算机硬件需要操作系统和应用软件
- DVD 播放器 + DVD 光盘
- 游戏机 + 游戏卡带

> 系统竞争的关键洞察：元件供应商不仅要关注同行的竞争，还要关注互补品厂商的配合。

### 转换成本与锁定效应

**转换成本**（switching costs）——从一种技术系统切换到另一种所需付出的成本。当转换成本非常高时，消费者被**锁定**(lock-in)：

| 锁定来源 | 例子 |
|---------|------|
| 学习成本 | 学会 Windows 后换 Mac 需要重新学习操作 |
| 数据格式 | 保存为 .docx 的文件无法被旧版 Word 打开 |
| 互补品投资 | 买了 PS5 的游戏光盘→换 Xbox 就全废了 |
| 社交网络 | 你的朋友都在微信上→你不能一个人用 Telegram |

对消费者来说，锁定是坏消息。但对供应商是好消息——**被锁定的消费者需求弹性极低**，可以抬高价格。

### 网络外部性

**网络外部性是外部性的一种特殊形式——一个人得自某种商品的效用取决于消费这种商品的其他人数量。**(范里安 Ch36)(V)

直接网络外部性：传真机——只有其他人也有传真机时你的传真机才有用。
间接网络外部性：录像机——对录像带的需求取决于录像机的数量，对录像机的需求又取决于录像带的数量。

**正反馈（positive feedback）** 是网络外部性的核心动力：
- 更多用户 → 更有价值 → 更有价值 → 更多用户 → …
- 这解释了为什么 VHS 打败了技术上更优的 Betamax
- 也解释了为什么 Windows 主导了 PC 操作系统

> **正反馈的关键后果：市场倾向于"赢家通吃"。**

### 双边市场（Two-Sided Markets）

某些平台服务于两类不同的用户群体，两边的需求相互依赖：

| 平台 | 群体 A | 群体 B | 交叉补贴方向 |
|------|--------|--------|------------|
| 谷歌搜索 | 用户（免费） | 广告商（付费） | 用广告商的钱补贴用户 |
| 信用卡 | 持卡人（低费） | 商户（手续费） | 用商户的钱补贴持卡人 |
| Uber | 乘客 | 司机 | 双边增长互相促进 |
| 游戏机 | 玩家 | 游戏开发商 | 补贴硬件→普及玩家→吸引开发商 |

双边市场的核心策略：**哪边的需求对价格更敏感，哪边就补贴。**

> 范里安在谷歌担任首席经济学家期间对双边市场和广告拍卖有深入研究。谷歌不向搜索用户收费，因为用户对"免费"极其敏感；而广告商对每次点击收费的敏感度较低。

### 常见误解

- ❌「网络效应=技术优胜」→ ✓ VHS 比 Betamax 差，但仍赢了。网络效应和市场历史比技术本身更能决定结果。
- ❌「锁定效应=用户傻」→ ✓ 在转换成本高的市场，理性用户也会被锁定——集体转换需要协调，协调本身是巨大的成本。
- ❌「免费=做慈善」→ ✓ 免费是一种定价策略。谷歌免费搜索 → 积累用户 → 卖广告 → 赚回来。

## 实战中的反复出现模式

### 模式 1：iOS vs Android——网络外部性的不对称竞争

苹果封闭生态（硬件+操作系统+应用商店）vs 谷歌开放生态（Android+Google Play 免费授权）。苹果的用户被锁定在 iOS 生态（高转换成本），因此苹果可以收取 30% 的应用商店抽成。Android 开放授权 → 低转换成本 → 难以锁定用户 → 谷歌只能靠广告赚钱。**锁定效应决定商业模式的选择。**

### 模式 2：团购/外卖平台的双边市场难题

美团在早期同时补贴商户（免佣金）和用户（满减优惠）。一旦两边都达到临界规模，平台开始"收割"（提高商户抽成、减少用户补贴）。双边市场的关键问题是"先突破哪一边"——通常先补贴弹性高的一边（用户），等用户规模到了再吸引商户。

### 模式 3：ChatGPT 的网络效应

大语言模型具有独特的网络效应：**更多用户 → 更多反馈数据 → 模型更好 → 更多用户。** 这就是 OpenAI 选择免费提供 ChatGPT 的逻辑：通过零价格快速积累用户，利用用户的使用数据改进模型。这不是慈善——这是典型的网络外部性策略。

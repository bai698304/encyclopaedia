# 文化研究与霍尔编码/解码

> 意义不是由发送者"传递"的——它是在接收者的解读中被"生产"出来的。——Stuart Hall, 1980(V)

## What：本质定义

你看完一部主旋律电影，觉得"这故事还行"。你旁边的朋友说"这不就是宣传吗？"另一个朋友说"虽然有点刻意，但出发点可以理解"。同样的电影，三种完全不同的反应。

斯图尔特·霍尔的编码/解码模型（Encoding/Decoding Model）就是用来解释这个现象的。他的核心论点：**信息的生产者（编码）和消费者（解码）不共享同一个意义系统——编码想要传递的意思，和解码实际接受到的意思，可能是完全不同的东西。**[格里芬Ch26(V); Hall, 1980(V)]

霍尔提出了三种受众解读立场：[格里芬Ch26(V); bestpolarbear 霍尔的三种受众解读模式 jijidown(V)]

1. **主导-霸权式解读（Dominant-Hegemonic）**：受众完全按照编码者的意图理解信息——你看到了电影想让你相信的东西，你接受了。例如观众看完国庆献礼片后完全认同"国家越来越强大"的叙事。

2. **协商式解读（Negotiated）**：受众在总体接受编码框架的同时，对某些具体细节做出修正——"大方向是对的，但这个细节太夸张了"。绝大多数人的解读属于这种——不完全接受，也不完全拒绝。

3. **对抗式解读（Oppositional）**：受众完全理解编码者的意图，但选择了完全相反的解读——"我知道你想让我信什么，但我偏不信，而且我有我的解释"。例如一个反战的观众看战争宣传片，每一帧都在解构它。（→ 参见06-批判学派与法兰克福学派的受众观）

**文化研究的目标：** 它不研究"媒体对受众做了什么"（效果范式），也不研究"受众如何满足自己的需求"（使用与满足）——它研究的是**权力如何通过文化生产不平等、以及被压制者如何抵抗**。权力和抗争是文化研究的两个核心词。[韦斯特Ch21(V)]

## Why：为什么重要

- **把受众从"文化工业的被动受害者"中解放出来**：与法兰克福学派不同，霍尔认为受众不是被动的文化傀儡。他们可以用自己的方式"读"媒体信息——甚至读出一个完全相反的版本。**对抗式解读的存在本身就是对文化工业的抵抗。**
- **引入了"霸权"概念**：葛兰西的霸权理论解释了为什么被压迫阶级不反抗——不是因为被强迫，而是因为**他们"同意"了**。统治阶级不是用武力压制，而是通过文化（媒体、教育、宗教）让被压迫者接受现有的秩序"就是正常的"。[韦斯特Ch21(V)]
- **将阶级分析扩展到了种族、性别、年龄**：传统马克思主义只关心"工人阶级vs资本家"，文化研究把被压迫的定义扩展到有色人种、女性、LGBTQ+、儿童——任何被主流叙事边缘化的群体。

## How：如何运作

**编码/解码链条：**

```
生产者（政治/媒体/广告）
  → 编码（将意识形态嵌入信息中）
    → 传播（信息通过媒体渠道流通）
      → 解码（受众在自身社会语境中解读）
        → 三种结果（主导/协商/对抗）
```

但这个链条不是线性的——编码和解码是两个相对自主的过程，它们之间没有必然的因果联系。信息的"意义"不是在发送端确定的，而是在接收端完成的。霍尔明确反对Shannon-Weaver的线性传播模型，他称之为"回路"（circuit）而非"链条"。[Hall, 1980(V)]

关键点在于：**编码和解码是两个相对独立的过程。** 信息的意义不是在"生产完成"时确定的，而是在"被解读完成"时才最终实现。同一个信息在不同社会位置（阶级、种族、性别、代际）的人那里，可能产生完全不同的意义。

**霸权——文化研究最核心的概念：** [韦斯特Ch21(V)]
葛兰西问了一个根本问题：**为什么人们在明知政治安排违背自己利益的前提下，仍然接受甚至支持它？** 答案不是暴力镇压——是文化和意识形态的"同意制造"。学校、媒体、宗教机构不是直接告诉你"你该服从"，而是让你觉得"现状就是正常的"。当你觉得"穷人之所以穷是因为不够努力"时，霸权就成功了——你不是被强迫相信这个的，是自愿的。

**文化研究的"接合"（articulation）概念：** 霍尔和同事们提出了一个重要的方法论概念——接合。意思是：不同的社会力量（阶级、种族、性别议题）可以"连接"在一起，形成一个临时的联盟。"环保"可以和"反资本主义"接合，也可以和"爱国主义"接合——取决于谁在接合、怎么接合。[刘海龙Ch10(V)]

## Where / When：适用边界

### 何时有效
- **分析受众的多样性反应**：为什么同一部政治宣传片，有人热血沸腾，有人嗤之以鼻，有人无感——编码/解码框架直接解释了三者的区别
- **理解"文化战争"的根源**：当不同群体用完全不同的框架解读同一个事件时（如"黑人的命也是命"运动中的媒体报道），文化研究揭示了这不是"误解"，而是不同的解码立场在冲突
- **边缘群体的话语权斗争**：女性、LGBTQ+、少数民族通过创造自己的媒介（独立媒体、自媒体）来对抗主流媒体的边缘化——这是对抗式解读的延伸

### 何时失效
- **高估了受众的主动反抗能力**："对抗式解读"需要受众对编码机制有充分认识——这不是每个人都具备的能力。文化研究有时候夸大了普通人的批判意识。大多数人的解码是协商式，而非对抗式——对抗需要成本（信息、知识、勇气）。
- **"霸权"概念的解释循环问题**：被压迫者不反抗→因为霸权让他们同意→怎么知道是霸权在起作用→因为他们不反抗。这种论证容易陷入循环——任何一个保守的选择都可以被解释为"霸权在起作用"，不可证伪。
- **低估了商业文化的复杂性**：文化研究把一切大众文化都看作"统治阶级的工具"，但商业文化中也有进步性——女性超级英雄电影既有消费主义（卖周边），也有性别平等信号。文化研究的二元框架（霸权vs抵抗）可能过于简化。
- **中国语境中的"官方/民间"双重解码**：中国受众面对同一信息源（如新闻联播），其解码不是单一的"主导-协商-对抗"——而是**分场域解码**：在正式场合说官方语言，在私下用朋友圈/群聊表达真实看法。这不是霍尔框架中的"协商"（折中），而是一种**场域分离**——同一个人的解码可以在两个场域中完全不同，且不冲突。
